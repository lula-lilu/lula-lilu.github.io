Github Pages + Hexo 快速搭建个人博客（windows）

Posted on 2016-08-27   |   In 工具

记录一下搭建过程，以免自己忘记
具体详细步骤有很多相关博客写过，所以也就不一步一步的记录下来，只是把主要步骤加上自己走弯路的地方记录一下。

  • Hexo的安装过程其实三行代码就可以搞定，只是环境搭建以及过程中会碰到各种问题，保持耐心和细心，其实走弯路的时候也学到了新的东西。
  • 另外就是发现学习软件最好的教程其实就是官方文档+搜索工具。附上hexo官方教程。
  • 一次只解决一个问题

注册Github账号

首先去官网GIthub注册一个帐号。
然后建立一个username.github.io的仓库（repository）

安装Github for windows

这一步之前看到各种不同的方法，很多人是直接安装git,还有一些说要配置SSH key之类的，其实直接安装Github for windows 即可完成。只需要傻瓜式软件安装，登陆自己之前注册的用户名即可。这里注意一下下载过程可能比较慢，建议翻墙。
